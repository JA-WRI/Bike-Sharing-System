## Running the Project
1. Clone repo
2. Suggested to open front-end folder in VScode
3. run 'nmp i'
4. run 'npm run dev'
5. Suggested to open backend folder in Intellij
6. make sure you have java 21 (configure jdk if needed)
7. Go to VeloMtl.java
8. Run project in intellij
